Como funciona o salvamento?

Suas alterações são salvas diretamente no seu navegador. Clique em "Salvar no Navegador" e seus dados estarão aqui quando você reabrir a página. Use "Exportar para Arquivo" para baixar um arquivo .html com os dados atuais.
